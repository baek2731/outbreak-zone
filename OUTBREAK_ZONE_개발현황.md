## 1. 프로젝트 개요

**장르:** 브라우저 기반 탑다운 로그라이크 서바이벌
**컨셉:** 지뢰찾기 메카닉 기반 병원체 회수 + 좀비 공포 서바이벌
**출품 목표:** CrazyGames

### 핵심 게임플레이 루프
```
기지 출발 (Space)
→ 소나(F)로 병원체/좀비 탐지
→ 병원체 회수 미니게임 (E + 방향키)
→ 좀비 조우 시 전투 미니게임 (F 연타)
  → 80% 게이지 시 치료제 선택지 (Y/N)
→ 전체 회수 후 출구 탈출 or 기지 복귀(F)
→ DNA로 기지 업그레이드
→ 다음 런
```

## 2. 세계관 (설계 문서 기준)

### 핵심 설정
- 생명공학 실험에서 생겨난 악성 개체(크리쳐)가 인류 말살을 목표로 함
- 크리쳐가 인간을 감염시켜 감염자(좀비)를 만들어냄
- 플레이어는 병원체 회수 요원 — 무기 없이 회수 도구와 치료제만 보유

### 두 종류의 적
| 구분 | 정체 | 처리 방식 |
|------|------|----------|
| 크리쳐 | 인공 생명체 | 전투 승리 → 스턴 4초 / 치료제 → 무효 + 새 크리쳐 스폰 |
| 감염자 | 감염된 인간 (전사자) | 전투 승리 → 스턴 / 치료제 → 안식 + DNA 보너스 |

### 최종 보스
5층 이후 지능형 크리쳐 ORIGIN — 엑스트라 스테이지 대상

### 엔딩
- **감염사:** 보라 오버레이 페이드인 → ☣ 좀비 전환 패널
- **ALL CLEAR:** 터미널 타이핑 (ORIGIN 기밀 문서) → 빨간 눈 등장 → TO BE CONTINUED

## 4. 키 매핑

| 키 | 기능 |
|----|------|
| 방향키 | 이동 |
| F | 소나 (꾹 눌러 차징 → 뗌으로 발사) |
| G | 정밀 소나 (진영/인식표 식별) |
| E | 병원체 회수 미니게임 시작 |
| D | 치료제 자가 사용 (감염 -10%) |
| Y | 전투 중 치료제 투여 (80% 선택지) |
| N | 전투 계속 (선택지 거부) |
| Space | 기지 출발 / 재탐사 / 다음 층 |
| F | 기지 복귀 (출구 팝업) |

**모바일:** 조이스틱(이동) + F/G/E/D 버튼, 전투 시 Y/N 버튼 자동 전환

## 5. 주요 시스템 현황

### 5-1. 소나 시스템 ✅
- 기본 소나: 병원체 핑 + 좀비 레이더 (미니맵)
- 정밀 소나: 좀비 진영 식별 `[C]`빨강 / `[I]`보라 + 인식표 표시
- 식별 정보는 이후에도 유지됨

### 5-2. 병원체 회수 미니게임 ✅
- 방향키 패턴 입력
- 성공: 병원체 +1, 감염 +N% (감염저항 특성으로 감소)
- 실패: 감염 +15%, 소음 발생 → 좀비 SEARCH

### 5-3. 전투 미니게임 ✅
- F 연타로 게이지 채우기
- 꾹 누름 방지 (e.repeat 체크)
- **80% 도달 시 치료제 선택지 (3초)**
  - Y: 치료제 투여 (감염자→안식/크리쳐→무효)
  - N: 계속 싸우기
- 승리: 좀비 스턴 4초 (번개 아이콘 페이드아웃)
- 패배: 산소 손실 + 감염 증가

### 5-4. 치료제 시스템 ✅
- 초기 1개 / D키 자가 사용 (감염 -10%)
- 전투 80% 선택지에서 Y로 투여
- 감염자+Y: 안식 성공 + DNA +5 + 좀비 소멸
- 크리쳐+Y: 무효 + 전투 실패 처리 + 새 크리쳐 강제 스폰
- HUD `💉 N` 표시

### 5-5. 전사자 시스템 ✅ (방법A - 튜토리얼 대체)
- 감염사 시 전사자 풀(`outbreak_fallen_pool`) 등록
- 증원 좀비 스폰 시 전사자 풀 있으면 감염자, 없으면 크리쳐
- **UNIT-00 강제 등록** (튜토리얼 완성 후 이 코드 제거 필요)
  - `ensureUnit00Fallen()` in `startGame()`
- 층당 최대 3명 제한

### 5-6. 패트롤 시스템 ✅
- 1단계: 좀비 증원 (공지 알림)
- 2단계: 이동속도 ×1.2
- 3단계: 이동속도 ×1.4 + 시야 확대

### 5-7. 산소/감염 시스템 ✅
- 산소 부족 → 감염도 증가
- 감염 100% → 좀비화 엔딩
- 캡슐 회수로 산소 회복

### 5-8. 기지 업그레이드 ✅
- DNA로 스테이터스 6종 / 특성 4종 강화
- 감염저항: Lv1 30% / Lv2 50% 감소율

### 5-9. 모바일 지원 ✅
- 터치 감지 자동 (1024px 미만 + touchstart)
- 조이스틱 (이동) + 액션 버튼 (F/G/E/D)
- 미니게임 중 D-PAD 자동 전환
- 전투 선택지 시 Y/N 버튼 자동 전환
- DEV 토글로 PC에서 테스트 가능

### 5-10. 사운드 시스템 ✅
- 효과음 25종 + BGM 7종 (MP3 파일 기반)
- zombie_chase 1~3 랜덤 재생 (2초 쿨타임)
- 채집 키 입력음 (collect_key.mp3)
- 인게임 볼륨 패널 (좌하단) + DEV 패널 개별 슬라이더
- 볼륨 기본값: master 0.25 / BGM 0.30 / SFX 0.75

### 5-11. 플로팅 팝업 + 공지 시스템 ✅
- 캐릭터 주변 플로팅 텍스트 (수치 변동)
- 화면 중앙 상단 공지 (패트롤 단계, 전체 회수 완료)

### 5-12. 엔딩 연출 ✅
- 감염사: 보라 오버레이 → ☣ 좀비 전환 패널
- ALL CLEAR (5층): 터미널 타이핑 → ORIGIN 빨간 눈 → TO BE CONTINUED 패널
- 빨리감기: Space/클릭 누르면 10배속

```javascript
// MG (미니게임 파라미터)
combatChoiceGauge:  80    // 치료제 선택지 등장 게이지

mineFailInfect:     15    // 회수 실패 시 감염 증가(%)

// CONFIG.serum
initialCount:       1     // 런 시작 치료제 수
selfHealAmount:     10    // 자가 사용 감염 감소(%)
choiceTime:         3.0   // 선택지 표시 시간(초)
infectedDnaBonus:   5     // 감염자 안식 DNA 보너스

// CONFIG.fallen
maxPerStage:        3     // 층당 최대 전사자 수
```
