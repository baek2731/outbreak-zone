# OUTBREAK ZONE — 슬롯 & 유품 계승 시스템 설계서

> 2026-06-30 대화에서 확정된 메타 시스템 설계.
> 본게임(스테이지 1~5) 밸런스 확정 후 구현 착수 권장.

---

## 1. 요원 슬롯

**기본 구조**
- 슬롯 4개 보유
- 각 슬롯은 독립된 요원 (코드네임, 강화 상태 개별 관리)
- DNA는 전체 공유, 강화는 요원별 개별 적용

**출격 제한**
```
감염도 0~49%   → 출격 가능
감염도 50% 이상 → 출격 불가, 회복 대기
```

**회복 단위**
```
런 1회 = 회복 1단계
(실제 회복량은 밸런스 테스트 후 확정)
```

**설계 의도**
- A 요원이 감염도 높아 못 쓰면 B/C/D로 전환 → 메타 레이어 자원관리
- "방금 산소 다 쓰고 감염도 70% 찍은 요원을 바로 재투입"하는 비현실성 해소
- 전사자 시스템과 결합 시 감정적 무게감 강화 (내가 키운 요원이 적으로 등장)

## 2. 사망 패널티

**전투사 (크리쳐에게 사망)**
```
강화 1티어 강등
→ 전사자풀 등록 → 해당 층 감염자로 등장
```

**감염사 (감염도 100%)**
```
강화 2티어 강등
→ 전사자풀 등록 → 해당 층 감염자로 등장
→ 세계관: 동료가 감염체가 됨 (더 무거운 패널티)
```

**설계 의도**
- "전부 소실"보다 "강등" 방식 채택 → 죽음에 패널티는 있지만 복구 의지가 남음
- 감염사가 전투사보다 무거운 패널티 → 산소/감염 관리 동기 강화

## 3. 유품 시스템

**유품 생성 조건**
```
필드에서 내 요원(감염자)과 조우
→ 정밀소나(G)로 식별 후 안식(Y) → 온전한 유품
→ 식별 없이 소멸            → 훼손된 유품 (계승 능력치 50%)
```

**일반 감염자와의 차이**
```
일반 감염자 안식 → DNA 보너스만 드랍
내 요원 안식    → DNA 보너스 + 유품 드랍
```

**유품 안내**
```
안식 시 글귀 등장: "ㅇㅇㅇ의 유품이 이 구역 어딘가에 있을 것이다."
→ 플레이어가 직접 탐색해서 회수해야 함 (자동 획득 아님)
```

**층당 유품 슬롯**
```
층당 슬롯 1개
새 유품 생성 시 → 기존 유품 유지, 새 유품은 스택 뒤로 적재
플레이어 방문 시 → 스택 맨 앞 유품만 필드에 등장, 획득 가능
획득 후 → 다음 런에 스택 다음 유품이 슬롯으로 올라옴
```

**유품 지속성**
```
탈출 실패 / 못 찾고 탈출 → 다음 런 동일 층에 유지 (소실 없음)
소멸로 생성된 훼손 유품   → 동일하게 유지, 계승 시 50%만 적용
```

**설계 의도**
- 정밀소나 미사용 시에도 유품을 통한 "두 번째 기회" 부여 → 완전 손실 방지
- 식별 여부에 따른 차등(온전/훼손) → 정밀소나(G)의 가치를 정보 도구에서 자원관리 도구로 격상
- 층당 1개 제한 → 인지 부하 최소화, 탐색 목표 명확화

## 4. 기지 계승 처리

**흐름**
```
필드에서 유품 획득
→ 기지 복귀 시 스택에 적재
→ 플레이어가 원하는 시점에 계승 처리
→ 대상 요원 선택 → 능력치 이전
```

**계승 범위**
```
온전한 유품 → 해당 항목 강화 1티어 추가
훼손된 유품  → 해당 항목 강화 0.5티어 (누적 후 정수 도달 시 적용)
```

**기지 스택 예시**
```
ECHO   — 훼손유품 (1층)
CIPHER — 온전유품 (2층)
NOMAD  — 온전유품 (2층)
```

## 5. 치료제 용도 재정리

```
필드 전용
├── 자가 감염도 회복 (D키)
├── 일반 감염자 안식 (Y키) → DNA 보너스
└── 내 요원 안식 (Y키)    → DNA 보너스 + 유품 드랍

기지 전용
└── DNA 소모 → 유품 계승 처리
```

**설계 의도**: 치료제 용도를 필드/기지로 분리해 희소성 분산 방지

---

## 6. 정밀소나(G) 가치 변화

```
기존: 크리쳐 / 감염자 식별 → 치료제 낭비 방지
신규: 내 요원 여부 식별    → 온전유품 / 훼손유품 분기
```

## 7. 스테이터스/특성 vs DNA변이 — 역할 분리

**문제 인식**: 현재 LOBBY.status/trait는 전부 %보정 수치형이라 체감이 약함(밋밋함). 플레이 자체의 스릴은 있으나 강화 효과가 눈에 띄지 않음.

**해결 방향**: 스테이터스/특성은 의도적으로 밋밋하게 유지(안정적 기초 성장), 그 위에 **DNA변이**라는 임팩트 레이어를 얹는다.

**명칭**: "유물" → "DNA변이" (세계관 톤에 맞춤. 병원체/감염/ORIGIN 설정과 일치)

**컨셉**: 두 가지 결 중 "인간을 벗어난 힘 + 디메리트" 채택
```
❌ 초과학기술 — 디메리트 없는 단순 파워업 (게임 정체성과 불일치)
✅ DNA변이    — 힘을 얻는 대신 무언가를 잃는다 (치료제 이중구조와 같은 철학)
```

**설계 원칙**
- 모든 변이가 디메리트를 가질 필요는 없음 — 희귀도 낮은 변이는 순수 버프, 희귀도 높은 변이는 트레이드오프
- 기존 status/trait 수치형 강화와 카테고리를 다르게 가져가 역할 중복 방지
- 현재 game.js의 실제 시스템과 맞물리는 효과여야 함 — "채집속도 증가" 같은 시스템에 없는 개념은 배제 (채집 미니게임은 패턴 입력 방식이라 '속도' 개념 자체가 없음)

**효과 설계 축 (상세 수치/목록은 추후 개발)**
```
회수 계열 — 정밀소나/채집(mine) 미니게임 연동
  예) 패턴 길이 감소, 실패 시 페널티 감소

생존 계열 — 산소/감염
  예) 산소 최대치, 드레인 감소, 감염 임계값 조정

전투 계열 — combat 미니게임
  예) F연타 게이지 효율 상승, 좀비 drain 감소

탐지 계열 — 소나/은신
  예) 소나 범위, 좀비 감지 회피, 소음 반경 감소
```

각 축에서 순수버프/트레이드오프를 섞어 변이 풀을 구성. 유품 계승 시스템(섹션 3~4)과 연결하면 "죽은 동료의 DNA변이를 물려받는다"는 서사로 자연스럽게 합쳐짐.

## 8. 요원 영입 (가챠 대안)

**가챠 방식의 우려**
- CrazyGames 환경(5~15분 세션)에서 확률형은 이탈 유발 가능성 높음
- 캐주얼 게임 특성상 확정 교환 방식이 더 적합

**추천 방향: 확정 교환**
```
DNA N → 특정 코드네임 요원 영입 가능
예) DNA 150 → NOMAD 영입 (이동속도 +10%, 소나범위 -1)
   DNA 200 → STALKER 영입 (감염저항 +20%, 이동속도 -5%)
```
- 전임자 코드네임(CIPHER, NOMAD, DRIFT 등)을 영입 가능 요원으로 활용 → 세계관 연결
- 진짜 가챠를 원할 경우 천장 시스템 필수

---

## 9. 미확정 항목 (밸런스 테스트 후 확정)

- 감염도 회복량 (런 1회당 몇 % 회복)
- 훼손 유품 0.5티어 누적 처리 방식
- 슬롯 초기 개방 수 (처음부터 4개 or 순차 개방)
- 요원 영입 방식 최종 확정 (확정교환 단독 or 가챠 병행)
- DNA변이 상세 효과 목록 및 희귀도 체계

---

## 10. 구현 순서 권장

```
0단계 — 본게임(1~5층) 밸런스 확정 (선행 필수)
1단계 — 슬롯 구조 + 출격 제한 + 회복 시스템
2단계 — 사망 패널티 (강등 로직)
3단계 — 유품 생성 + 층 슬롯 관리
4단계 — 기지 계승 처리 UI
5단계 — DNA변이 상세 설계 + 적용
6단계 — 요원 영입
```
